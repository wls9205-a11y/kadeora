## 0. 핵심 판단: 왜 이 물건이 중요한가

### 타이밍 프리미엄
- 분양가 미정 상태 = "예측" 콘텐츠가 가장 많은 관심을 끄는 구간
- 공고 후에는 모든 플랫폼이 동일 정보를 다룸 → 차별화 불가
- **D-24인 지금이 콘텐츠 골든타임**

### 지역 특성
- 구포동 = 부산 북구에서 신축 공급이 극히 드문 지역
- 호갱노노 30,216회 방문 + 433명 알림 → 이미 수요 검증됨
- 부산 북구 apt_sites 중 카더라에 등록된 활성 물건 = 0 (이 물건이 첫 번째)

### 우리만의 무기
- 구포동 실거래 데이터 보유 (반도유보라 74㎡ 4.6~5.5억, 동원로얄듀크 85㎡ 4.1~4.9억)
- 47개 블로그 자동 생성 크론 (apt, subscription, builder, district 등)
- 109개 호스팅어 사이트 백링크 네트워크
- Solapi 알림톡 + 웹푸시 인프라 가동 중
- consultant_leads 테이블 + 상담사 프로필 시스템 이미 구축

## 1. 내가 빠뜨렸던 것들

### 1-1. 지역주택조합 리스크 콘텐츠 (가장 큰 누락)

"두산위브 트리니뷰 구명역"은 **지역주택조합** 사업임. 일반 분양과 근본적으로 다른 리스크 구조:
- 토지확보율 미달 시 사업 무산 가능
- 추가분담금 발생 리스크
- 사업 지연 (실제로 이 물건도 5월→6월→다시 5월 조정됨)
- 조합원 vs 일반분양 물량 구분

**왜 중요**: "지역주택조합 위험성", "지역주택조합 장단점" 등은 검색 볼륨이 높은 키워드. 이 리스크를 솔직하게 다루면 신뢰도가 폭증하고 타 부동산 사이트와 차별화됨. 대부분의 분양 관련 사이트는 장점만 나열함.

**콘텐츠 안**: "두산위브 트리니뷰 구명역, 지역주택조합이라 주의할 점 5가지 — 토지확보율부터 추가분담금까지"

### 1-2. 검색 키워드 전략 (상위 키워드 선점)

"두산위브 트리니뷰 구명역"은 아직 검색 볼륨이 낮음. 진짜 트래픽이 오는 키워드:
- "구포동 분양" / "구포동 신축 아파트"
- "부산 북구 청약" / "부산 북구 분양"
- "구명역 아파트" / "구명역 역세권"
- "부산 지역주택조합"
- "구포동 시세" / "구포동 아파트 가격"

**전략**: 블로그 10편을 이 키워드들로 분산 배치. "두산위브 트리니뷰" 정확 키워드 + 상위 키워드 양쪽 다 잡기.

### 1-3. 비교 페이지 활용 (기존 인프라 낭비)

`/apt/compare/[slugs]` 페이지가 이미 있음. 이걸 안 쓰면 기능 낭비:
- 두산위브 트리니뷰 vs 리버파크반도유보라 vs 구포동원로얄듀크비스타
- 분양가(예측) vs 주변 실거래가 시각화
- 각 단지별 교통/학군/생활 편의 비교

### 1-4. 기존 크론 자동 연동 (자동 콘텐츠 생성)

apt_sites에 등록만 하면 기존 크론들이 자동으로 콘텐츠를 생성해줌:
- `blog-apt-v2`: 분양 정보 블로그 자동 생성
- `blog-subscription-strategy`: 청약 전략 콘텐츠
- `blog-builder-analysis`: 두산건설 시공 분석
- `blog-district-guide`: 구포동/북구 지역 가이드
- `blog-competition-rate`: 경쟁률 분석 (공고 후)
- `blog-upcoming-projects`: 예정 분양 리스트에 자동 포함
- `apt-ai-summary`: AI 분석 텍스트 자동 생성
- `indexnow-new-content`: 발행 즉시 구글/네이버 인덱싱 요청

### 1-5. FAQ 리치 스니펫 (구글 노출 극대화)

apt_sites의 `faq_items` 필드 → JSON-LD FAQPage 스키마로 자동 렌더링 → 구글 검색 결과에서 FAQ 아코디언 표시. 예시:
- "두산위브 트리니뷰 구명역 분양가는 얼마인가요?"
- "청약 일정은 어떻게 되나요?"
- "지역주택조합 아파트의 장단점은?"
- "구명역에서 도보 몇 분인가요?"
- "구포동 주변 시세는 어떻게 되나요?"

이걸 넣으면 검색 결과 페이지에서 카더라가 차지하는 면적이 3배로 늘어남.

### 1-6. 에버그린 콘텐츠 (분양 후에도 트래픽)

분양 정보는 시한성 콘텐츠. 분양 끝나면 트래픽 0. 이걸 방지하려면:
- "구포동 살기 어떤가요? — 교통, 학군, 생활편의 총정리" (동네 가이드)
- "구명역 역세권 도보생활권 완벽 분석" (교통 분석)
- "부산 북구 아파트 시세 추이 — 5년간 데이터로 본 투자 가치" (데이터 분석)

이건 분양과 무관하게 검색 유입이 계속됨.

### 1-7. OG 이미지 최적화 (네이버/카카오 공유)

`/api/og?title=...&category=apt&design=2` 시스템으로 자동 생성되지만, 이 물건 전용 OG를 더 매력적으로:
- 분양가 예측 범위 + 주변 시세 비교를 OG 이미지에 넣기
- 네이버 모바일 630x630 (`/api/og-square`) 최적화
- "D-24 모집공고 임박" 긴급성 표시

### 1-8. 실거래가 변동 알림 (분양 전후 추적)

구포동 새 실거래가가 등록되면 자동 알림:
- 분양 전: "주변 시세 이만큼 올랐습니다 → 분양가 예측 상향"
- 분양 후: "분양가 대비 주변 시세 갭 분석"
- `price_alerts` 테이블 + `apt-price-change` 크론 연동

### 1-9. 부산 유저 기반 현실 인식

현재 부산 실유저 18명, 전체 실유저 90명. 이건 현실적으로:
- 내부 커뮤니티 활성화보다 **SEO 트래픽 유입이 절대 우선**
- 토론방/피드 활성화는 시드유저로 분위기 조성 후 유입 유저가 참여하는 구조
- 상담사 연결은 아직 시기상조 → 대신 "상담 문의" 폼으로 리드 수집 + 추후 연결

### 1-10. 구포동 주변 단지 기존 데이터 교차 활용

우리 DB에 이미 있는 구포동 거래 데이터:

| 단지명 | 면적 | 준공 | 최근 거래가 | 평당가 |
|--------|------|------|------------|--------|
| 리버파크반도유보라 | 75㎡ | 2021 | 4.60~5.48억 | 약 2,000만/평 |
| 구포동원로얄듀크비스타 | 85㎡ | 2014 | 4.10~4.92억 | 약 1,600만/평 |
| 구포현대 | 78~85㎡ | 1994 | 2.20~2.84억 | 약 900만/평 |

→ 이 데이터를 `price_comparison` 필드에 넣으면 상세 페이지에서 자동 비교 차트 표시

### Phase C — 멀티채널 증폭 (Day 3~14)

#### C-1. 호스팅어 109개 사이트 교차 발행
- 부동산/재테크 관련 사이트 중 10~15개 선별

### Phase D — 유저 획득 퍼널 (Day 7~)

#### D-1. BlogReadGate 연동

#### D-2. 청약 알림 CTA (전용)
- apt_sites 상세 페이지 최상단: "모집공고 알림 받기" 버튼
- 클릭 → 비로그인 시 카카오 로그인 유도 → `apt_alerts` 테이블에 저장
- 모집공고일에 알림톡 자동 발송

#### D-3. 상담 리드 수집 (단순 폼)
- 상세 페이지 하단: "청약 상담 문의하기" 버튼

- 현재 상담사 연결은 보류, 리드 수집만 먼저 (추후 부산 북구 부동산 파트너 확보 시 연결)
