```

      주식 SEO 최적화 부족.
```

```
          blog-trade-analysis, blog-competition-rate, blog-monthly → 6개
  주식: blog-stock-v2, blog-stock-cluster, blog-stock-deep,
        blog-etf-compare, blog-adr-compare, blog-sector-rotation → 6개

동등한데 콘텐츠 비율이 83:14 → 주식 크론의 생산량이 낮음

해결: 주식 크론 실행 빈도 상향
  blog-stock-v2: 3시간 → 2시간
  blog-stock-cluster: 6시간 → 3시간
  blog-stock-deep: 6시간 → 3시간

목표: 주식 블로그 비율 14% → 30%

```

```
  "삼성전자 관련 분석 블로그 3편"
- 블로그 내 종목 링크 → 내부 순환 트래픽 증가
- 주식 비교 페이지 (/stock/vs/) 콘텐츠 확대

```

```
conversion_events에 variant 컬럼 추가:
  event_type, cta_name, variant ('A' | 'B'), page_path, visitor_id

1주 후 CTR 비교 → 승자 전체 적용
```

**어드민 모니터링:**
```

  Variant B: 노출 84 → 클릭 3 (3.6%) ← 승자
```
