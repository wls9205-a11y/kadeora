## 제출 방법

### 네이버 웹마스터도구
1. https://searchadvisor.naver.com → 로그인
2. 사이트 관리 → kadeora.app 선택 (미등록 시 사이트 추가)
3. 요청 → 웹 페이지 수집 → 아래 URL 하나씩 입력
4. 1일 최대 10개까지 제출 가능 → 10일에 걸쳐 제출

### 구글 Search Console
1. https://search.google.com/search-console → 로그인
2. 속성 선택 → kadeora.app (미등록 시 속성 추가)
3. URL 검사 → 아래 URL 입력 → "색인 생성 요청"
4. 1일 최대 ~50개 가능
