## 0. 왜 다시 설계하는가

기존 어드민은 **데이터를 보여주는 대시보드**였다. 숫자는 있는데 **"그래서 뭘 해야 하는가"**에 대한 답이 없었다. 새 설계는 단순한 모니터링이 아니라 **"이번 주에 집중해야 할 한 가지"를 제시하는 운영 코파일럿**이다.

## 1. 새 구조: 6개 탭

```
┌──────────────────────────────────────────────────────────────┐
│  🎯 집중    📊 성장    👤 사용자    🗄️ 데이터    🔧 운영    ⚡ 실행  │

```

| 탭 | 핵심 질문 | 기존에 없던 것 |
|---|---|---|
| **🎯 집중** | "이번 주 가장 중요한 한 가지는?" | AI 기반 주간 진단 + 자동 우선순위 |
| **📊 성장** | "성장하고 있나, 죽어가고 있나?" | 퍼널 시각화 + 기능 사용 히트맵 |
| **👤 사용자** | "유저들이 뭘 하고 있나?" | 관심단지/워치리스트 현황 + 리텐션 코호트 |
| **🗄️ 데이터** | "콘텐츠가 건강한가?" | 데이터 신선도 타임라인 + 품질 점수 |
| **🔧 운영** | "고장난 곳은?" | 크론 그룹별 헬스 + 리소스 |
| **⚡ 실행** | "버튼 하나로 최신화" | 🚀 전체 최신화 + 스마트 실행 |
