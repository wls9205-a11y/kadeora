## 0. 왜 대형 이벤트인가 — 자산 관점

**일반 블로그글**: 발행 직후 피크 → 3개월 후 감가
**대형 이벤트 블로그 묶음**: 6~10년 단계별 트래픽 재점화 · **복리 자산**

| 단지 사례 | 이벤트 규모 | 예상 누적 PV (5년) | 발행 노력 |
|---|---|---|---|
| 레이카운티 (기존) | 단일 이슈 (재분양) | ~50,000 | 1편 수동 |
| 삼익비치 (제안) | 7단계 라이프사이클 | **~500,000** | 10편 허브+스포크 |

10편 네트워크가 단일 글 대비 **10배 누적 효율**. 후속 대형 이벤트 30개에 템플릿 재적용하면 **15배~30배**.

## 1. 라이프사이클 × 의도 매트릭스

### 재건축 7 Stage × 검색 피크 시점

| Stage | 단계 | 피크 키워드 | 네이버 월검색량 추정 | 삼익비치 현재 |
|---|---|---|---|---|
| 1 | 논의·추진위 | "XX 재건축 언제" | 낮음 | ✗ (지났음) |
| 2 | 조합설립 | "조합 가입", "매수 타이밍" | 중 | **현재 근접** |
| 3 | 안전진단·사업시행 | "진행상황 2026", "시공사" | 높음 | **← 현재** |
| 4 | 관리처분·이주 | "이주비", "분담금", "조합원 분양가" | 높음 | 임박 |
| 5 | 철거·착공 | "착공식", "평형 확정" | 중 | ~2027 |
| 6 | **일반분양 (MAX)** | "청약", "분양가", "경쟁률" | **최고** | ~2028 |
| 7 | 입주·준공 | "입주 후기", "실거래가", "전세" | 중~높 (장기) | ~2032+ |

**핵심 통찰**: Stage 6이 전체 트래픽의 40%+ 발생. **Stage 6 대응 글을 2년 미리 발행** = SEO 적립 → 개화 시 독점.

| 유형 → \ 의도 ↓ | What | When | Price | Compare | How | News |

| 관심지역민 | ✓ | ✓ | - | - | - | ✓ |
| 현 조합원 | - | ✓ | ✓ | ✓ | ✓ | ✓ |
| 투자자 | ✓ | ✓ | ✓ | ✓ | - | ✓ |

- naver_syndication → 네이버 본 블로그 발행 시 **카더라 링크 명시**

- 평당가/상승률 변동 시 본문 숫자 + 차트 자동 갱신 + `updated_at` 신선도 유지
- 네이버는 **최근 수정된 글**에 가산점
