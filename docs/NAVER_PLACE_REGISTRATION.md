- 대표 이메일: kadeora.app@gmail.com

## 등록 절차

### Step 1. 업체 등록
1. [ ] https://smartplace.naver.com 접속
2. [ ] 네이버 계정으로 로그인 (기존 검색광고 계정 있으면 재사용)
3. [ ] "내 업체 등록" 클릭
4. [ ] 카테고리: 부동산·금융정보 플랫폼 / 전문·과학·기술 서비스
5. [ ] 주소: 사업자등록 주소 그대로 (등기부 일치)
6. [ ] 소개 문구: "대한민국 소리소문 부동산·주식 정보 커뮤니티 — 청약, 실거래, 종목 분석을 하나의 앱에서."

### Step 2. 소유권 인증
1. [ ] 사업자등록증 사본 업로드 (JPG/PDF)
2. [ ] 네이버 검색광고 계정 연동 (선택이나 권장)

### Step 3. 미디어/링크
1. [ ] 로고 이미지 업로드 (`/icons/icon-192.png`, `/icons/icon-512.png`)
2. [ ] 대표 이미지 3~5장 (/public/og-image* 또는 스크린샷)
3. [ ] 홈페이지 URL: https://kadeora.app
4. [ ] 네이버 블로그 URL: https://blog.naver.com/kadeora (Step 5 에서 개설)
5. [ ] 전화번호 (고객센터): 공용 번호 또는 대표이사 번호

### Step 4. 네이버 검색 등록
1. [ ] https://submit.naver.com/add.naver 에서 사이트 등록
2. [ ] "블로그 URL" 탭에서 카더라 공식 네이버 블로그 등록 (연관 콘텐츠)
3. [ ] https://searchadvisor.naver.com 웹 마스터 도구 소유 확인 (docs/NAVER_SC_SETUP.md)

### Step 5. 네이버 블로그 개설 (권위 신호)
1. [ ] https://blog.naver.com 에서 ID `kadeora` 또는 `kadeora_app` 확보
2. [ ] 블로그 프로필: 사업자등록번호·홈페이지 역링크
3. [ ] 주 2회 포스트 계획 (scripts/naver-blog-sync.ts 로 자동화 후보)

### Step 6. 추가 외부 신호 (선택)
- [ ] 위키미디어 공통데이터에 회사 엔트리 (장기)
- [ ] 카카오맵 장소 등록 (별도)
- [ ] 구글 비즈니스 프로필 등록 (별도)
- [ ] 네이버 지식iN 공식 기업계정 신청 (장기)

## 검증 (등록 후 2~6주)
1. 네이버 통합검색 "카더라" 쿼리 → 스마트플레이스 카드 노출
2. `site:kadeora.app` 네이버 검색 → 주요 경로 색인
3. 네이버 지도에서 상호 검색 → 위치 카드 노출 (주소 기반)
4. 네이버 검색 "카더라 주식", "카더라 청약" → 브랜드 확장 검색 추적

## 실패 케이스 대응
- 소유권 인증 반려 → 사업자등록증 원본 업로드 재시도
- 카테고리 부적합 → "정보통신·기타" 로 전환
- 상호 중복 경고 → "카더라 kadeora.app" 으로 변경 등록
