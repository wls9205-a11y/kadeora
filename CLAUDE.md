## 2. 피드 카드 (PostCard / FeedClient) — 토스 스타일로 전면 재설계

### 현재 vs 목표

현재: 제목 크고 볼드 + 본문 + 카테고리pill + 좋아요/댓글/공유 아이콘 과다
목표 (토스 스타일):

```
┌──────────────────────────────────────┐
│ 🟢 닉네임        등급뱃지   · 2시간 전│  ← 1행: 아바타 + 닉네임 + 등급 + 시간
│                                      │
│ 글 본문 텍스트가 여기에 표시됩니다.    │  ← 본문 (제목 없이 본문만, 3줄 클램프)
│ 두 번째 줄 텍스트...                  │     제목이 있으면 볼드로 첫 줄에
│                                      │
│ ♡ 5    💬 3    ↗                     │  ← 인터랙션: 좋아요 + 댓글 + 공유 (3개만)
└──────────────────────────────────────┘
 ─────────── 가는 구분선 ───────────────
```

### 구체적 변경사항

```
카드 레이아웃:
- padding: px-4 py-3
- 카드 간 구분: border-b border-slate-800 (다크) / border-slate-100 (라이트)
- 배경: transparent (카드 배경 없음, 리스트형)
- 그림자: 없음
- 좌측 보더: 없음 (인기글 보더 제거)

1행 (작성자 정보):
- 좌측: 아바타 36px 원형 (해시컬러 + 닉네임 첫글자)
- 아바타 우측: 닉네임 (font-medium text-sm) + 등급 아이콘(작게)
- 우측: "· 2시간 전" (text-xs text-slate-400)
- 카테고리 pill 제거 (피드에서는 보이지 않음, 글 상세에서만)

본문:
- 제목이 있으면: font-semibold text-[15px] 첫 줄
- 본문: text-[14px] text-slate-300 (다크) / text-slate-600 (라이트)
- 최대 3줄 클램프 (line-clamp-3)
- 이미지 썸네일: 본문 아래에 rounded-xl (우측 배치 아님)

인터랙션 바:
- 좌측 정렬, gap-5
- ♡ + 숫자 (Heart 아이콘 18px)
- 💬 + 숫자 (MessageCircle 18px)  
- ↗ (Share2 18px, 숫자 없음)
- 색상: text-slate-400, liked시 Heart만 text-red-500 fill
- 북마크 아이콘: 제거 (⋯ 메뉴 안으로 이동)
- 공유 텍스트 "공유": 제거 (아이콘만)

⋯ 더보기 메뉴 (선택사항):
- 카드 우상단에 MoreHorizontal 아이콘
- 클릭 시: 저장(북마크) / 신고 / 차단
```

## 3. 글 상세 페이지 (/feed/[id]) — 토스 스타일

### 현재 문제
- "댓글 3" + "댓글 3" 중복 표시
- 댓글 입력이 스크롤 중간에 위치
- 하단 액션바가 단순

### 목표
```
┌──────────────────────────────────────┐
│ ← 뒤로                              │
├──────────────────────────────────────┤
│                                      │
│ 🟢 닉네임  등급  · 8시간 전          │
│                                      │
│ 미분양 아파트 매수 전 체크리스트      │  ← 제목 (text-lg font-bold)
│                                      │
│ 미분양 매수 전 반드시 확인할 것들:    │  ← 본문 (text-[15px] leading-relaxed)
│ 1. 미분양 원인 분석                  │
│ 2. 시행사/시공사 재무 상태           │
│ ...                                  │
│                                      │
│ ⚠️ 투자 면책 (부동산/주식만)         │
│                                      │
├──────────────────────────────────────┤
│ 댓글 3                               │  ← "댓글" 섹션 헤더 (1개만!)
├──────────────────────────────────────┤
│ 🟢 닉네임1 · 1시간 전               │
│ 댓글 내용                            │
│ ─────────────────────────────────── │
│ 🔵 닉네임2 · 30분 전                │
│ 댓글 내용2                           │
├──────────────────────────────────────┤
│                                      │
│  (여백)                              │
│                                      │
╞══════════════════════════════════════╡
│ ♡ 6   💬 3         🔗공유  🔖저장   │  ← 하단 고정 액션바
╞══════════════════════════════════════╡
│ [댓글 입력...]                   ↑   │  ← 하단 고정 댓글 입력
└──────────────────────────────────────┘

변경사항:
- "● 댓글 3" + "● 댓글 3" 중복 → "댓글 3" 1개만
- 댓글 입력: 하단 고정 (position: sticky bottom-0)
- 액션바: 댓글 입력 바로 위에 고정
- 전송 버튼: 둥근 원형 (bg-brand) + ArrowUp 아이콘
```

## 4. 글쓰기 (/write) — 토스 스타일 극심플

### 현재 → 목표
```
현재: 카테고리 선택 + 제목 + 본문 + 다양한 UI
목표 (토스 스타일):

┌──────────────────────────────────────┐
│ ← 뒤로                    남기기     │  ← 우상단에 제출 버튼
├──────────────────────────────────────┤
│                                      │
│ (선택) 제목을 입력해주세요           │  ← placeholder, 선택사항
│ ─────────────────────────────────── │
│                                      │
│ 내용을 입력해주세요...               │  ← 본문 textarea, 넓게
│                                      │
│                                      │
│                                      │
│                                      │
│                                      │
│ ⚠️ 광고, 비난, 도배성 글을 남기면    │  ← 경고 (text-xs text-slate-400)
│ 영구적으로 활동이 제한될 수 있어요.  │
│                                      │
├──────────────────────────────────────┤
│ 📷사진  📊카테고리                   │  ← 하단 도구바 (최소한)
└──────────────────────────────────────┘

변경사항:
- 카테고리 선택을 상단에서 하단 도구바로 이동
- 제목 필드를 선택사항으로 (placeholder: "(선택) 제목을 입력해주세요")
- 본문 영역을 최대한 넓게
- 자본시장법 경고 → 간결한 1줄 경고로 축소
- "남기기" 버튼을 우상단에 (토스 스타일)
```

## 5. 사이드바 정리 (데스크톱)

### 현재 문제
- 인기 검색어 + 등급 안내 + 가이드북 + 사업자 정보 → 과밀

### 변경
```
우측 사이드바:
1. 인기 검색어 → 유지하되 간결하게 (7개→5개)
2. 등급 안내 → 접힌 상태 (기본 숨김, 클릭 시 펼침)
3. 가이드북/사업자 정보 → 푸터로 이동
4. "알림 받기" 배너 → 제거 또는 매우 작게

목표: 사이드바가 피드보다 시선을 뺏지 않도록
```

## 6. 바텀 네비 — 토스 스타일

```
현재 5개 탭: 피드 / 부동산 / + / HOT / 토론
+ 버튼이 가운데에 크게 있음

변경:
- + 버튼 크기 줄이기 (현재 너무 큼)
- 또는 + 를 제거하고 피드 상단에 글쓰기 프롬프트만 유지
- 아이콘 + 텍스트 레이블 (text-[10px])
- 현재 탭: brand 컬러
- 나머지: text-slate-500
```

## 작업 순서

```
1. FeedClient.tsx: 인기 글 좌측 보더 제거
2. PostCard.tsx / FeedClient.tsx: 피드 카드 토스 스타일 재설계
   - 카테고리 pill 제거 (피드에서)
   - 인터랙션 바 간소화 (좋아요/댓글/공유 3개만)
   - 카드 배경/그림자 제거, border-b만
   - 북마크→⋯메뉴 안으로 이동
3. feed/[id]/page.tsx: 글 상세 재설계
   - 댓글 중복 제거
   - 하단 고정 액션바 + 댓글 입력
4. write/WriteClient.tsx: 글쓰기 극심플화
5. 사이드바 정리
6. 각 단계 npm run build → 커밋 → push
```

## Claude Code 시작 프롬프트

```
소스코드를 먼저 전부 읽어:
cat src/app/(main)/feed/FeedClient.tsx
cat src/components/shared/PostCard.tsx
cat src/app/(main)/feed/[id]/page.tsx
cat src/app/(main)/write/WriteClient.tsx

읽은 다음, 위 작업 순서대로 진행해:
1. 인기 글 좌측 주황 보더 완전 제거
2. 피드 카드 토스증권 스타일로 전면 재설계 (카테고리pill 제거, 인터랙션바 간소화, 카드배경/그림자 제거)
3. 글 상세 댓글 중복 제거 + 하단 고정 액션바/댓글입력
4. 글쓰기 극심플화 (제목 선택사항, 본문 넓게, 우상단 남기기 버튼)
5. 사이드바 정리 (등급안내 접기, 가이드/사업자 푸터 이동)

각 단계 npm run build 확인 후 커밋+푸시. 논스톱으로 끝까지.
```
