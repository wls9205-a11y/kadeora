## 완료된 주요 기능
### 코드
- NoticeBanner: 헤더 아래 초록 텍스트 전광판 (site_notices 테이블)
- GuestCTA: 비로그인 유저 회원가입 유도 배너
- OneLinerSection: 게시글 한줄평 (comment_type='oneliner')
- PWAInstallTracker: standalone 모드 자동 설치 기록
- RefreshButton: 피드 새로고침 버튼 + 햅틱
- lib/haptic.ts: iOS AudioContext + Android vibrate 통합 햅틱
- AdminCharts.tsx: SVG 라인차트 + 도넛차트 (외부 라이브러리 없음)
- AptClient: 청약/미분양 탭 전환, 미분양 지역별 섹션, 상태 배지 색상
- 청약 가점 진단 페이지: /apt/diagnose
- 하단 탭바 중앙 글쓰기 플로팅 버튼 (주황 +버튼)
- 공유 API: /api/share (share_logs 기록)
- 공지 미리보기: NoticeManager 실시간 미리보기
- 푸시 발송: 전체/웹/앱 구분 + 4기기 미리보기(Android/iPhone/Chrome/Safari) + 이미지 첨부
